# Installer Sources

## Microsoft Store

- **PhotoScape X:** https://apps.microsoft.com/detail/9nblggh4twwg?hl=en-US&gl=US - Quick photo editor
- **PowerToys:** https://learn.microsoft.com/en-us/windows/powertoys/ - Extra Windows features


## External Windows Tweaks

- **7+ Taskbar Tweaker:** https://ramensoftware.com/7-taskbar-tweaker - Make the Taskbar much more usable. Windows 10 only, seemingly.
- **Light Switch:** https://github.com/joakimmag/Light-Switch/releases - Light/Dark theme toggle
- **Msys2:** https://www.msys2.org/ - For compiling C/C++ software
- **PathCopyCopy:** https://pathcopycopy.github.io/ - Get UNIX style paths in context menu, and other Explorer path options
